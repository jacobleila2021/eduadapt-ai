Sample lesson for testing EduAdapt AI
=====================================

File: sample_lesson.docx

This is a Grade 6 Earth Science lesson on the water cycle. It includes
learning objectives, instruction, practice, and assessment so analytics
and AI adaptations have realistic content to work with.

How to get the file
-------------------
1. Double-click run.bat in the project folder (creates it automatically), OR
2. Run: python create_sample_lesson.py

How to test the app
-------------------
1. Start the app with run.bat
2. Upload samples/sample_lesson.docx
3. Review Lesson Analytics
4. Click Generate Adaptations (requires OPENAI_API_KEY in .env)
