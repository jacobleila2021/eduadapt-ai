# LCE Composition Contracts

LCE is the only layer allowed to construct lesson prose.

## Narrative voice

- Expert classroom teacher + NCERT textbook quality
- Never ChatGPT tone, never metadata dumps
- Forbidden tells: “delve”, “dive into”, “as an AI”, “let’s explore”

## Paragraph rules

- One idea per paragraph
- Prefer 2–5 sentences; avoid one-sentence fragments unless intentional
- Keep under ~120 words

## Required lesson flow

Hook → Concept teaching → Worked example → Practice → Summary → Revision → Reflection → Application

## Concept teaching (never skip)

Concept → Simple explanation → Real-life example → Visual → Worked example → Common misconception → Practice → Reflection

## STEM integrity

Inject `EngineResult` / official answers unchanged. Never invent equations, balances, or scientific diagrams.

## Vocabulary card contract

Every card: term (Capitalized, bold, centered) · pronunciation · part of speech · definition · simple explanation · example · picture · synonyms · antonyms · related concepts · difficulty · reading level · color

## Visual contract

- UVIE / verified STEM visuals only
- Place immediately after the explanation they support
- Prefer professional SVG; Mermaid only if explicitly requested

## Adaptive contract

Each version intentionally rewritten for its learner. Accessibility improves learning without reducing curriculum depth.
