# LCE Adaptive Writing Guide

## Principle

Never recolor the same lesson. Rewrite structure, scaffolds, and voice for the learner.

| Version | Intentional difference |
|---------|------------------------|
| standard | Full progressive classroom teaching |
| ld | Chunked steps, bold terms, calm pacing (same depth) |
| dyslexia | Short lines, bold key terms, spacing cues |
| adhd | 2-minute bursts, checkpoints, movement breaks |
| autism | Predictable map, literal language, explicit “Now/Next” |
| ell | Glossary scaffolds + sentence frames; keep board terms |
| visual | Diagram-first emphasis, colour-coded stages |
| auditory | Say:/Repeat: prose paragraphs |
| teacher | Differentiation map + teaching notes |
| parent | Plain-language home coaching |
| vocabulary | Premium CLG flashcards only |
| worksheet | Exam paper from CLG assessments/facts |

Accessibility **improves** learning. It must not simplify concepts below curriculum requirements.
