# LCE Teaching Rules

## Progressive teaching

Move from familiar experience → precise classroom language → practice → application.

## Scaffolding & chunking

- Neurodiversity / ADHD / Dyslexia lenses chunk into short steps
- Autism lens uses predictable section maps and literal language
- ELL keeps board vocabulary with glossary scaffolds — never baby-talk curriculum away

## Educational storytelling

Open with a clear goal, teach with examples, address misconceptions, close with reflection and application.

## Subject arcs (reuse packs)

| Subject | Arc |
|---------|-----|
| Mathematics (MIP) | Concrete → Visual → Representation → Symbols → Worked Example → Practice → Application |
| Physics (PIP) | Concept → Phenomenon → Experiment → Diagram → Formula → Example → Practice |
| Chemistry (CIP) | Concept → Particle view → Reaction → Equation → Diagram → Safety → Application |
| Biology (BIP) | Concept → Process → Diagram → Labels → Analogy → Application |
| English (ELIP) | Reading → Vocabulary → Grammar → Writing → Speaking → Listening → Literature |
| Social Science (SSIP) | Timeline → Map → Cause-effect → Primary source → Citizenship → Inquiry |
| Computer Science (CSIP) | Algorithm → Flowchart → Code → Memory → Execution trace |
| Commerce (CEIP) | Scenario → Table → Graph → Case study → Decision |
| World Languages (WLIP) | Pronunciation → Sentence building → Culture → Listening → Speaking |

## No isolated fragments

Every heading has teaching prose beneath it. No repeated sentences. No disconnected metadata lists taught as content.
